# Chapter 15 Labs: Deployment & Operations

This directory contains lab exercises for Chapter 15: Deployment & Operations.

## Lab Overview

The labs in this chapter focus on practical implementation of Deployment & Operations concepts using the Unitree G1 robot and ROS 2 Humble.

## Labs Included

1. **Lab 15.1**: Introduction to Deployment & Operations - Basic Concepts
2. **Lab 15.2**: Intermediate Deployment & Operations - Implementation
3. **Lab 15.3**: Advanced Deployment & Operations - Integration and Optimization

## Prerequisites

- Docker and docker-compose
- Basic ROS 2 knowledge
- Understanding of Deployment & Operations theory (Chapter 15)

## Getting Started

Each lab has its own directory with:
- Dockerfile for the lab environment
- ROS 2 nodes implementing Deployment & Operations concepts
- Launch files for easy execution
- Documentation and assessment criteria

To run a specific lab:
1. Navigate to the lab directory
2. Follow the instructions in the README.md file
3. Use the provided Makefile for common operations

## Assessment

Each lab includes assessment criteria to validate your implementation of Deployment & Operations concepts.
