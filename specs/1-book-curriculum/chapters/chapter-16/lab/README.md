# Chapter 16 Labs: Capstone Project

This directory contains lab exercises for Chapter 16: Capstone Project.

## Lab Overview

The labs in this chapter focus on practical implementation of Capstone Project concepts using the Unitree G1 robot and ROS 2 Humble.

## Labs Included

1. **Lab 16.1**: Introduction to Capstone Project - Basic Concepts
2. **Lab 16.2**: Intermediate Capstone Project - Implementation
3. **Lab 16.3**: Advanced Capstone Project - Integration and Optimization

## Prerequisites

- Docker and docker-compose
- Basic ROS 2 knowledge
- Understanding of Capstone Project theory (Chapter 16)

## Getting Started

Each lab has its own directory with:
- Dockerfile for the lab environment
- ROS 2 nodes implementing Capstone Project concepts
- Launch files for easy execution
- Documentation and assessment criteria

To run a specific lab:
1. Navigate to the lab directory
2. Follow the instructions in the README.md file
3. Use the provided Makefile for common operations

## Assessment

Each lab includes assessment criteria to validate your implementation of Capstone Project concepts.
