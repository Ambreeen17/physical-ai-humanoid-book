# Chapter 7 Labs: Motion Planning

This directory contains lab exercises for Chapter 7: Motion Planning.

## Lab Overview

The labs in this chapter focus on practical implementation of Motion Planning concepts using the Unitree G1 robot and ROS 2 Humble.

## Labs Included

1. **Lab 7.1**: Introduction to Motion Planning - Basic Concepts
2. **Lab 7.2**: Intermediate Motion Planning - Implementation
3. **Lab 7.3**: Advanced Motion Planning - Integration and Optimization

## Prerequisites

- Docker and docker-compose
- Basic ROS 2 knowledge
- Understanding of Motion Planning theory (Chapter 7)

## Getting Started

Each lab has its own directory with:
- Dockerfile for the lab environment
- ROS 2 nodes implementing Motion Planning concepts
- Launch files for easy execution
- Documentation and assessment criteria

To run a specific lab:
1. Navigate to the lab directory
2. Follow the instructions in the README.md file
3. Use the provided Makefile for common operations

## Assessment

Each lab includes assessment criteria to validate your implementation of Motion Planning concepts.
