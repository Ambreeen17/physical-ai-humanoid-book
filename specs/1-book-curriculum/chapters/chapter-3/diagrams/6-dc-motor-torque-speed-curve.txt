================================================================================
                    MOTOR EQUATIONS
================================================================================

    VOLTAGE EQUATION
    ═════════════════

    V = E + I·R

    Where:
        V = Applied voltage (V)
        E = Back EMF (V) = K_e · ω
        I = Armature current (A)
        R = Armature resistance (Ω)


    TORQUE EQUATION
    ═══════════════

    τ = K_t · I

    Where:
        τ = Electromagnetic torque (Nm)
        K_t = Torque constant (Nm/A)
        I = Armature current (A)


    BACK EMF RELATIONSHIP
    ═════════════════════

    E = K_e · ω

    Where:
        K_e = Back EMF constant (V·s/rad)
        ω = Angular velocity (rad/s)


================================================================================
    DERIVING THE TORQUE-SPEED RELATIONSHIP
    ══════════════════════════════════════

    From V = E + I·R:

    V = K_e · ω + (τ / K_t) · R

    Solving for ω:

    ω = (V / K_e) - (R / (K_e · K_t)) · τ

    Or in slope-intercept form:

    ω = ω_no-load - K · τ

    Where:
        ω_no-load = V / K_e        (no-load speed)
        K = R / (K_e · K_t)        (slope magnitude)
        τ_stall = (V · K_t) / R    (stall torque)


    RELATIONSHIP BETWEEN CONSTANTS
    ══════════════════════════════

    SI Units: K_e = K_t = K

    At steady state (P_in = P_out):
        V·I = τ·ω
        (E + I·R)·I = (K·I)·ω
        E·I + I²·R = K·ω·I
        K·ω·I + I²·R = K·ω·I
        I²·R = 0 (ideal, no losses)

    Thus: K_e = K_t = K (numerically equal in SI units)


================================================================================
                    POWER CURVES
    ══════════════════════════════════════════════════════════════════════

    Speed (ω)
      ↑
      │    ╱╲ Power Output
      │   ╱  ╲____
      │  ╱        ╲____
      │╱              ╲____
      ├──────────────────────────────► Torque (τ)
     τ_stall          τ_max   τ_opt
                           (Peak Power)


    POWER CALCULATIONS
    ══════════════════

    Output Power:     P_out = τ · ω
                     P_out = τ · (ω_no-load - K·τ)
                     P_out = τ·ω_no-load - K·τ²

    Maximum Power:    dP_out/dτ = 0
                     τ_peak = ω_no-load / (2K) = τ_stall / 2

    Input Power:      P_in = V · I
    Efficiency:       η = P_out / P_in = (τ·ω) / (V·I)

================================================================================
                    GEAR REDUCTION EFFECTS
    ═══════════════════════════════════════════════════════════════════════

    WITHOUT GEARBOX                 WITH GEARBOX (N:1)
    ═══════════════                 ════════════════════

    Speed:  ω_motor                 Speed: ω_load = ω_motor / N
    Torque: τ_motor                 Torque: τ_load = τ_motor × N × η_gear

    ┌──────────────────┐            ┌──────────────────┐
    │   ╱╲             │            │           ╱╲     │
    │  ╱  ╲            │            │          ╱  ╲    │
    │ ╱    ╲           │     N:1    │         ╱    ╲   │
    │╱      ╲          │ ─────────► │        ╱      ╲  │
    ├──────────────►   │            │       ╱         ╲ │
    │   τ_stall             │            │      │          │
    │                      │            │      ▼          │
    │                      │            │   ╱╲            │
    │                      │            │  ╱  ╲           │
    │                      │            │ ╱    ╲          │
    │                      │            │╱      ╲         │
    │                      │            │────────►        │
    │                      │            │   N×τ_stall     │
    └──────────────────────┘            └──────────────────┘

    Benefits:
    - Increased torque at output (×N)
    - Decreased speed at output (/N)
    - Reduced motor heating (less current at same load torque)
